WooCommerce Table Rate Shipping - Define Table Rate Shipping rates based on weight, item count, class and price..

== Description ==

This is a WooCommerce Table Rate Shipping extenstion that can set multiple shipping rates based on weight, item count and price.

**HOW TO USE THE PLUGIN**

[youtube https://youtu.be/5brJQXgNVE0]

**FEATURES**

* DEFINE RULES based on WEIGHT (kg, lbs, grams, oz).
* DEFINE RULES based on QUANTITY (min-max)
* DEFINE RULES by CLASS
* DEFINE CALCULATION TYPE (per order, rates per item, line item & per class)
* DEFINE CLASS PRIORITIES
* Can set Per row price
* Can set Per item cost
* Can set Custom shipping label
